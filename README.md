Developed a virtual healthcare system for depression detection using deep learning and machine learning techniques, including Convolutional Neural Networks (CNN) with Haar Cascade for facial emotion recognition and TF-IDF with Naive Bayes for sentiment analysis. The system integrates multimodal inputs such as facial expressions and text data to improve the accuracy of emotional detection.

Designed and implemented an emotion-aware chatbot to provide real-time mental health support through interactive and empathetic conversations. Enhanced user engagement by enabling continuous interaction and personalized responses based on detected emotional states.

Additionally, built a user-friendly interface to visualize emotional trends and provide personalized mental health insights. This project strengthened my skills in AI, NLP, computer vision, and full-stack integration while contributing to an innovative solution for accessible mental healthcare.
